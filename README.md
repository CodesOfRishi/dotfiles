# Dotfiles

**Using [`stow`](https://www.gnu.org/software/stow/) and `git` for managing and tracking my Linux configuration files.**

## Applying the Configurations

```bash
git clone https://github.com/CodesOfRishi/dotfiles.git $HOME/dotfiles/
cd dotfiles
stow */
```
